# 英语学习游戏

一个基于 React + Vite 构建的英语学习游戏，集成了 Supabase 后端服务，支持在线学习进度同步。

## 🚀 快速开始

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```
访问：http://localhost:5173/english-learning-game/

### 构建生产版本
```bash
npm run build
```

### 预览生产版本
```bash
npm run preview
```

### 部署到 GitHub Pages
```bash
npm run deploy
```

## 📂 项目结构
```
src/
├── components/      # React 组件
├── config/         # 配置文件
├── context/        # React Context
├── engines/        # 游戏引擎
├── hooks/          # 自定义 Hooks
├── lib/            # 工具库和 Supabase 配置
└── routes/         # React Router 路由
public/
├── audio/          # 音频文件
├── images/         # 图片资源
└── words/          # 单词图片
```

## 🛠 技术栈
- React 19
- Vite 8
- React Router 6
- Tailwind CSS
- Supabase
- ESLint
