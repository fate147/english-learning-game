# 题目生成规则

本文件定义各科目题目的生成规范，确保题目质量一致、难度合理。

---

## 通用规则

### 1. 难度分级（Concrete → Abstract）

| 阶段 | 说明 | 占比 |
|------|------|------|
| concrete | 具体直观，有图形/实物辅助 | 30% |
| pictorial | 图像化，半抽象表示 | 30% |
| abstract | 纯文字/数字，无视觉辅助 | 25% |
| synthesis | 综合应用，多步骤思考 | 15% |

### 2. 选项设计

- **选项数量**：3 个
- **干扰项原则**：
  - 必须是合理错误（不是随机数字）
  - 反映常见误区（如用错公式、计算错误）
  - 长度与正确答案相近（不要一眼看出）
- **正确答案位置**：随机分布在 A/B/C

### 3. 通用禁忌

- ❌ 不要出现与题目无关的干扰项
- ❌ 不要出现超出年级范围的知识点
- ❌ 不要出现单位不一致（如米和厘米混用）
- ❌ 不要出现歧义表述

## 英语题目规则

### 1. 对话题（Dialogue）

- **场景真实**：符合小学生日常生活场景
- **选项设计**：
  - 正确答案：语法正确、语义贴切
  - 干扰项：语法错误 或 语义不贴切（不要两者都错）
- **长度控制**：每个选项 3-8 个单词
- **词汇范围**：限于该年级已学词汇

### 2. 图片选择题（Image Choice）

- **图片匹配**：图片必须准确对应单词含义
- **干扰图片**：同类别但不同物品（如 apple → pear, banana）
- **避免抽象概念**：不要用图片表示抽象词汇

### 3. 字母填空题（Letter Fill）

- **空格数量**：单词长度的 40%-50%，至少 2 个，最多 5 个
- **干扰字母**：3-4 个，不能与正确字母重复
- **字母顺序**：打乱（Fisher-Yates 洗牌）

## 数学题目规则

### 1. 图形规范

#### 正方形
```
┌──────────┐
│          │
│          │  边长 Xcm
│          │
└──────────┘
```
- 宽度与高度字符数比例约 2:1（补偿字体宽高比）
- 标注"边长 Xcm"

#### 长方形
```
┌──────────────┐
│              │  Xcm
│              │
└──────────────┘
    Ycm
```
- 宽度明显大于高度
- 分别标注长和宽

#### 网格
```
┌──┬──┬──┐
│  │  │  │
├──┼──┼──┤
│  │  │  │
└──┴──┴──┘
  X行Y列
```
- 每格大小一致
- 标注行数和列数

### 2. 数值规范

- **整数优先**：避免小数，除非是小数单元
- **单位统一**：题目和选项单位一致
- **数值合理**：符合生活实际（如书包不会重 100kg）

### 3. 选项设计

- **正确答案**：使用正确公式/方法
- **干扰项类型**：
  - 用错公式（如面积用周长公式）
  - 计算错误（如 12×8=96 写成 98）
  - 概念混淆（如周长和面积混用）

## 语文题目规则

### 1. 阅读理解

- **文章长度**：100-200 字（三年级）
- **问题类型**：
  - 信息提取：文中直接能找到答案
  - 简单推断：根据上下文推断
  - 情感理解：感受人物心情
- **选项设计**：
  - 正确答案：简洁明确
  - 干扰项：曲解文意 或 张冠李戴

### 2. 字词题

- **生字范围**：该年级必学字
- **拼音规范**：声调标注正确
- **组词规范**：常用搭配，不要生僻词

### 3. 古诗/课文

- **填空内容**：关键实词，不要填虚词
- **理解题**：基于字面意思，不要过度解读

---

## 检查清单

生成新题目后，逐项检查：

- [ ] 难度分级是否合理？
- [ ] 选项数量是否为 3 个？
- [ ] 干扰项是否合理（不是随机）？
- [ ] 正确答案位置是否随机？
- [ ] 图形是否准确（正方形像正方形）？
- [ ] 数值/单位是否一致？
- [ ] 是否符合该年级知识范围？
- [ ] 表述是否清晰无歧义？
