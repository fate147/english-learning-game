# 语文「故事小侦探」功能设计

> 基于英语游戏已有框架，复用同一套机制，内容换成语文阅读理解。

---

## 一、整体定位

### 设计原则

- **不增加新规则**——英语怎么玩，语文就怎么玩
- **素材用课本课文**——练阅读 + 熟悉课文，一箭双雕
- **AI 动态出题**——同一篇课文可以出几十种不同理解题，永不重复
- **星星共用**——语文赚的星星跟英语通用，孩子不会觉得是"额外作业"

### 入口位置

```
主界面
  ├── [ 英语 ]  ← 已有
  ├── [ 语文 ]  ← 新增
  └── [ 数学 ]  ← 预留
```

---

## 二、游戏流程

每局 8 题，跟英语保持一致，孩子无需适应新规则。

```
进入语文
  ↓
选择单元 / 课文（可选"随机"）
  ↓
开始一局 → 每局 8 题，每题包含：
    1. 一篇短文（30-80字，取自课文或课文改编）
    2. 一个问题
    3. 3-4个选项
  ↓
每题作答后：
    ✓ 正确 → "破案成功！" 音效 + 得星
    ✗ 错误 → "再想想" + 错误停留 2 秒 + 展示正确答案
  ↓
8 题结束 → 结算星星 + 鼓励语
```

### 与英语的一致性

| 维度 | 英语 | 语文 |
|------|------|------|
| 每局题数 | 8 题 | 8 题 |
| 题型结构 | 4 选读 + 4 填字母 | 6 选择 + 2 填空（台阶递进） |
| 正确反馈 | 鼓励语 + 得星 | 鼓励语 + 得星 |
| 错误反馈 | 修正 + 停留 | 修正 + 停留 |
| 星星系统 | 兑换真实奖励 | **共用同一星星池** |

## 三、题型设计（4 个台阶）

### 第一阶：选故事标题（第 1-2 题）

> 最基础的概括——读完知道这篇文章在说什么。

**形式：** 展示短文 → 选最合适的标题

**例（课文《富饶的西沙群岛》节选）：**

```
短文：西沙群岛是南海上的一群岛屿。那里的海水五光十色，海底有
各种各样的珊瑚，鱼多得数不清。海滩上有美丽的贝壳，岛上还有茂
密的树林。

问题：这篇短文最合适的标题是什么？

A. 美丽的海岛
B. 富饶的西沙群岛 ← 正确答案
C. 海底世界
```

**能力训练：** 区分"描述内容"和"核心主题"

---

### 第二阶：选段落大意（第 3-4 题）

> 理解一段话的核心意思，区分"细节"和"主要意思"。

**形式：** 展示一段话 → 选最准确的大意

**例（课文《荷花》节选）：**

```
短文：荷花已经开了不少了。荷叶挨挨挤挤的，像一个个碧绿的大圆盘。
白荷花在这些大圆盘之间冒出来。有的才展开两三片花瓣儿。有的花瓣
儿全都展开了，露出嫩黄色的小莲蓬。有的还是花骨朵儿，看起来饱胀
得马上要破裂似的。

问题：这段话主要讲了什么？

A. 荷叶的样子
B. 荷花的不同姿态 ← 正确答案
C. 莲蓬很可爱
```

**能力训练：** 抓段落主旨，不被细节带偏

---

### 第三阶：推理理解（第 5-6 题）

> 答案不在原文里，需要结合上下文推断。这是从"读懂了字面"到"读懂了意思"的关键一步。

**形式：** 展示短文 → 推断原因、结果、人物情感

**例（课文《掌声》）：**

```
短文：英子犹豫了一会儿，慢吞吞地站了起来，眼圈红红的。在全班
同学的注视下，她终于一摇一晃地走上了讲台。

问题：英子为什么"眼圈红红的"？

A. 她生病了
B. 她紧张、害怕被嘲笑 ← 正确答案
C. 她刚才哭过
```

**能力训练：** 理解人物的情感和动机

---

### 第四阶：关键词填空（第 7 题）

> 从"选择"到"产出"，降低难度保留阶梯感。

**形式：** 给一句不完整的概括，填关键词

**例：**

```
句子：这段讲了西沙群岛海水___，海底有___和___，是个___的地方。

可选词：[五光十色] [珊瑚] [鱼] [美丽富饶]
```

**能力训练：** 从文中提取关键信息组织语言

---

### 第五阶：用自己的话说（第 8 题，Bonus 题）

> 最高难度，但孩子有前面 7 题的铺垫，不会感到无从下手。

**形式：** 语音/文字输入 → AI 判断核心要素是否点到

**评价标准（只检查三个点）：**
- ✅ 是否提到核心事件/主题
- ✅ 是否有自己的表达（不是照抄原文）
- ✅ 是否基本准确

**不论对错都给星**——敢开口说就是胜利。

> **注意：** 此题为加分性质，不对也不扣星。目的是鼓励孩子输出，而不是打击信心。

## 四、素材生成方案

### 核心策略

> 一篇课文 ≠ 一道题。一篇课文可以出无限道题。

每次进入游戏，AI 根据当前课文**动态生成**题目，所以没有"题做完了"的问题。

### 素材来源

| 方式 | 说明 |
|------|------|
| **手动录入** | 你把孩子当前的课本目录（单元+课文名）告诉我 |
| **拍照上传** | 拍一课课文发给我，我来提取内容 |
| **指定版本** | 你说版本（如部编版三年级下册），我按通用课文列表来 |

### 生成规则

每篇课文按以下维度出题，保证每题都不一样：

```
出题维度矩阵
├── 标题选择（全文/段落）
├── 段落大意（不同段落）
├── 细节理解（不同句子/关键词）
├── 推理（原因/结果/情感/意图）
├── 填空（不同关键词位置）
└── 难度控制（简单/中等/挑战）
```

### 防止重复

- 每次出题记录已出过的题号（保存在本地）
- 相同课文再次出题时，优先选未用过的类型和角度
- 所有题都出过一轮后，重置记录，从新角度再出

## 五、与英语系统的共用设计

### 星星系统（完全共用）

```
英语得星 → 存入总星星池
语文得星 → 存入总星星池（同一池）
    ↓
孩子用总星星兑换奖励
```

### 奖励库（不变）

已有的兑换项（买玩具、出去玩、多玩 15 分钟）保持不变。语文只是多了一个赚星星的方式，不会影响奖励系统的平衡。

### 界面风格

英语游戏的图标、配色、动效保持一致，只是入口文字从"English"变成"语文"。

## 七、数学「题目侦探」设计（CPA 教学法）

### 7.1 核心理念

> **做对题 ≠ 理解题。理解题 ≠ 理解数学。**

三年级孩子数学题做错，80% 不是不会算，是**没读懂题**。所以数学部分**不练计算，只练阅读理解**——把数学题当语文阅读理解来做。

### 7.2 科学方法：CPA 模型

国际上经过大量实证验证的教学方法，新加坡数学的核心。分成三步走：

```
Concrete（具体）  →  Pictorial（图像）  →  Abstract（抽象）
  动手摸实物          动手画图形          只看符号
```

#### 第一步：Concrete（具体）

用实物让孩子**摸到**数学，建立感性认识。

```
题目：3箱苹果，每箱12个，一共几个？

不出算式，给实物图/动画：
- 3个盘子，每个盘子里放12个苹果的图片
- 让孩子数

他数出来：12+12+12=36
你问："发现了什么？"
他说："三个十二加起来。"

→ 这时候他才真懂了"乘法的意义"
```

#### 第二步：Pictorial（图像）

用图形代替实物，从"动手"过渡到"动眼"。

```
还是"3箱苹果"，但变成图形条：

┌─────────────┐
│     12      │
└─────────────┘
┌─────────────┐
│     12      │
└─────────────┘
┌─────────────┐
│     12      │
└─────────────┘

问："用算式怎么表示？"
→ 3×12=36（他能说出来"3个12"，就是真懂）
```

#### 第三步：Abstract（抽象）

只看数字和符号，理解数量关系。

```
3 × 12 = 36

问孩子："3代表什么？"（箱数/份数）
        "12代表什么？"（每箱个数/每份数）
        "36代表什么？"（总数）
如果他都能说清楚，才算真正掌握。
```

### 7.3 "会做但说不明白"的根源

```
学校教：抽象 → 刷题（跳过前两步，直接给公式）
孩子：  明明不知道为什么，但必须做对 → 反感
```

**你的直觉是对的**——跳过理解和建模，直接跳进抽象符号里刷题，孩子当然反抗。

### 7.4 游戏中的题型设计（4 阶，8 题）

和英语、语文保持一致的 8 题 / 局模式。

#### 第一阶：看图识意（第 1-2 题）→ Concrete

> 展示实物图，选对应的算式。不计算，只建立"实物 → 符号"的连接。

```
┌──────────────────────┐
│  🍎 🍎 🍎 🍎 🍎     │
│  🍎 🍎 🍎 🍎 🍎     │  ← 2排苹果，每排5个
│  🍎 🍎 🍎 🍎 🍎     │
└──────────────────────┘

问题：上面这些苹果，用哪个算式表示？

A. 2 + 5 = 7
B. 3 × 5 = 15 ← 正确答案（3排，每排5个）
C. 3 + 5 = 8
```

**能力训练：** 从实物中抽象出"几个几"的数量关系

---

#### 第二阶：看图选故事（第 3-4 题）→ Pictorial

> 展示图形条（bar model），选对应的现实情境。反过来练——看到模型能联想到实际问题。

```
┌─────────────┐
│      ?      │
├──────┬──────┤
│  24  │  24  │
└──────┴──────┘

问题：上面的图形条，表示下面哪个故事？

A. 小明有24颗糖，分给2个朋友，每人几颗？
B. 小红有2盒饼干，每盒24块，一共多少块？ ←
C. 小明有24本书，又买了2本，一共几本？
```

**能力训练：** 理解"每份数 × 份数 = 总数"的模型结构

---

#### 第三阶：读懂题目（第 5-6 题）→ Abstract

> 纯文字应用题，不计算，选"这道题在问什么"或"需要哪两个数"。

**题型 A：选问题意图**

```
小明每分钟走 65 米，从家到学校走了 12 分钟，他家离学校有多远？

问题：这道题要我们算什么？

A. 65 个 12 加起来是多少 ← 正确答案
B. 12 个 65 加起来是多少（本质一样，故意混淆说法）
C. 65 和 12 哪个大
```

**题型 B：选关键信息**

```
妈妈买了 3 种水果，苹果 12 个，梨 8 个，橘子 15 个。
全家每天吃 5 个水果，够吃几天？

问题：要解决这个问题，需要用到哪几个数？

A. 12 + 8 + 15 = 35，35 ÷ 5 = 7 ← 正确答案
B. 12 + 8 = 20，20 - 5 = 15
C. 15 - 12 = 3，3 + 8 = 11
```

**能力训练：** 提取关键信息，排除干扰信息

---

#### 第四阶：选同类模型（第 7-8 题）→ 综合

> 最高阶——给一道题，选另一个情境不同但结构一样的故事。
> 这是判断孩子是否真懂"数学模型"的黄金题型。

```
妈妈买回 3 箱苹果，每箱 12 个，一共买了多少个？

问题：下面哪个故事跟这道题是"同一种算法"？

A. 小明有 5 个文具盒，每个里面放 8 支笔，一共多少支笔？ ←
B. 小明有 5 个苹果，吃了 2 个，还剩几个？
C. 小明有 8 颗糖，分给 4 个朋友，每人几颗？
```

**能力训练：** 识别"每份数 × 份数 = 总数"这一通用结构

### 7.5 与英语、语文的对比

| 维度 | 英语 | 语文 | 数学 |
|------|------|------|------|
| 每局题数 | 8 题 | 8 题 | 8 题 |
| 教学法 | 听/说/读 | 阅读理解五阶 | **CPA 三阶** |
| 题型 1-2 | 听音选图 | 选标题 | 看图选算式（Concrete） |
| 题型 3-4 | 字母填空 | 选段落大意 | 看图形选故事（Pictorial） |
| 题型 5-6 | 选句意 | 推理理解 | 读题选意图/关键信息（Abstract） |
| 题型 7-8 | 对话理解 | 填空+自己说 | 选同类模型（综合） |
| 正确反馈 | 鼓励语+星 | 鼓励语+星 | 鼓励语+星 |
| 错误反馈 | 修正+停留 | 修正+停留 | 修正+停留 |
| 星星 | 共用 | **共用同一池** | **共用同一池** |

### 7.6 难度控制

| 级别 | 适用 | 说明 |
|------|------|------|
| 简单 | 初次接触某类题 | 选项差异明显，干扰项逻辑偏差大 |
| 中等 | 有一定基础 | 选项接近，需要仔细分辨 |
| 挑战 | 熟练掌握 | 需要跨单元/跨学期知识识别模型 |

### 7.7 素材来源

- 同步教材：部编版三年级数学上下册
- 按单元切分：时分秒/万以内的加减/倍的认识/多位数乘一位数/长方形和正方形/分数的初步认识
- AI 动态生成：每次进游戏根据当前单元生成新题

## 八、进度追踪

### 可记录的数据

| 维度 | 说明 |
|------|------|
| 每课正确率 | 孩子对哪篇课文/哪个单元理解好 |
| 薄弱题型 | 概括弱 / 推理弱 / 细节弱 / 模型识别弱 |
| 星级走势 | 长期进步曲线，正向反馈给孩子看 |

### 应用

- 每周汇总一次，你看完后可以决定下周期重点练什么
- 不展示给孩子看（避免变成"成绩压力"）

## 九、总结

可记录孩子每篇课文的正确率，标记薄弱题型类型（概括弱 / 推理弱 / 细节弱），后续可以针对性强化。

## 九、总结

```
你有的：英语游戏（已验证，孩子愿意主动玩）
      + 星星兑换真实奖励系统
      + 孩子的抵触心理（最讨厌学校式刷题）

你想要的：语文阅读理解训练
        + 数学理解训练（不刷题，练读懂题目）
        + 都用课本内容当素材
        + 孩子不反感
        + 对学校作业有帮助

解决方案：不改游戏规则，只换内容入口。
        三个科目共用同一套游戏框架，
        孩子赚星星的途径多了两条，
        学校上课时发现"这个我在游戏里学过"。

核心教育理念：
  - 语文：阅读理解五阶法（选标题→选大意→推理→填空→自己说）
  - 数学：CPA 教学法（具体→图像→抽象）
  - 共同点：先理解，再输出；搭台阶，不跳级；游戏化，不刷题
```
